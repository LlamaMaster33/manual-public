## Compiling the Manual

To compile the LaTeX document into a PDF:

### Prerequisites

You need a LaTeX distribution installed on your system:

- **Linux**: `sudo apt-get install texlive-latex-base texlive-latex-extra`
- **macOS**: Install [MacTeX](https://www.tug.org/mactex/)
- **Windows**: Install [MiKTeX](https://miktex.org/) or [TeX Live](https://www.tug.org/texlive/)

### Compilation

```bash
pdflatex arc_foxtrot_manual.tex
pdflatex arc_foxtrot_manual.tex  # Run twice for proper cross-references
```

This will generate `arc_foxtrot_manual.pdf`.
